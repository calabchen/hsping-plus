# 快速开始

## 1. 目标

用最短路径跑通一次业务闭环：
建卷 -> 生成答题卡 -> 上传 -> 自动批改 -> 教师复核 -> 查看成绩

## 2. 访问入口

- Laravel API：`http://127.0.0.1:8000`
- Vue3 前端：`http://127.0.0.1:8001`
- Python 服务：按本地配置启动，不固定端口

## 4. 5 分钟体验

1. 登录系统
2. 进入试卷管理查看已有试卷
3. 进入批改管理查看提交记录
4. 打开成绩页查看统计

## 5. 标准操作流程

### 5.1 班级与学生
1. 创建班级
2. 添加学生或 CSV 导入

### 5.2 试卷与答题卡
1. 创建题目
2. 创建试卷并设置分值
3. 生成答题卡
4. 发布到班级

### 5.3 上传与批改
1. 上传答题卡（单张/批量）
2. 系统自动识别与判分
3. 教师复核主观题得分
4. 提交最终成绩

### 5.4 查看与导出
1. 按试卷查看总分与明细
2. 按班级查看均分与及格率
3. 导出 Excel

## 6. 常见问题

- 识别失败：先检查图片清晰度和完整度
- 客观题误判：在复核页手动修正
- 主观题不准：AI 仅建议分，以教师确认为准
