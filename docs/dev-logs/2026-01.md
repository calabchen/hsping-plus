## 📅 2026-01-07 | 完成登录界面（第1天）
### ✅ 已完成
- [✓] 在 `frontend-pc/src/views/auth/Login.vue` 搭建基础登录表单（邮箱、密码、提交按钮）。
- [✓] 接入 `authStore.login()`，统一调用后端 `POST /api/login`。
- [✓] 提交态处理：增加 `loading`，避免重复点击提交。

### 🛠 遇到问题
- **问题**: 登录失败时只有通用报错，无法快速判断是账号错误还是会话问题。CORS blocked
- **原因**：后端未放行 8001，且带 cookie 时不能用 allowed_origins=['*']
- **解决**:
  1. 先保留用户友好提示 `登录失败，请检查账户和密码。`。
  2. 在后续联调中通过 Network 面板定位具体 HTTP 状态码。
  3. CORS_ALLOWED_ORIGINS 加入 http://localhost:8001,http://127.0.0.1:8001;
SANCTUM_STATEFUL_DOMAINS 加入 localhost:8001,127.0.0.1:8001; config/cors.php 使用明确 origin 列表; 执行 php artisan optimize:clear 并重启服务

  3. .env 设置 SESSION_SAME_SITE=null。php artisan config:clear + 重启后端
